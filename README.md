# Discord-AutoCalBot
The first `Persian` calendar bot that changes its nickname and connects to a voice channel




## Setup
To run the bot, first, change the CONFIG class values in AutoCal.py file and for run 24/7 run via runner.py file
```python
class CONFIG:
    TOKEN = '' #str
    GUILD_ID = 0 #int
    CHANNEL_ID = 0 #int
```
