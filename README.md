# Discord-AutoCalBot
The first persian calendar bot that changes own nick name and connect to voice channel




## Setup
To run the bot, first, change the CONFIG class values in AutoCal.py file and for run 24/7 run via runner.py file
```python
class CONFIG:
    TOKEN = '' #str
    GUILD_ID = 0 #int
    CHANNEL_ID = 0 #int
```
